Cantina Reis App

Mobile E-commerce Project

App criado para agilizar o atendimento e diminuir as filas da cantina da universidade.

Desafio & Objetivo

O maior desafio para o desenvolvimento deste projeto foi aprender a utilizar React Native e Expo.

As etapas iniciais, desde as instalações, e a forma sincronizada de trabalhar com o pc e o celular foram empolgantes.

O principal objetivo deste projeto foi aprender e desenvolver um aplicativo mobile e-commerce para agilizar o atendimento da cantina da universidade, diminuindo as filas nos horários de pico.

01. UX Research e Levantamento de Requisitos

Pesquisas e reuniões abrangentes e detalhadas com a cliente e público-alvo (estudantes da universidade).

Essa etapa foi extremamente importante para entender as necessidades e expectativas da cliente e dos futuros usuários do aplicativo.

02. Business Model Canvas e Organograma

A partir dos dados coletados, foi criado o Business Model Canvas para visualizar de forma organizada os pontos principais do projeto.

A elaboração do primeiro organograma também foi feita nessa etapa.

03. Fluxo do Usuário e Wireframe

O mapa do Fluxo do Usuário auxiliou na estruturação da lógica do desenvolvimento.

Essa etapa é muito importante pois o aplicativo deve seguir um padrão de navegação simples e intuitiva para não gerar dúvidas ao usuário.

Wireframe: primeiros esboços para possibilidades do layout do aplicativo.

Foram feitas muitas alterações pois o número de telas necessárias para incluir todas as funcionalidades previstas era muito grande.

04. Protótipos de Alta Fidelidade

Na plataforma Figma, foram criados os Protótipos de Alta Fidelidade.

Para a primeira etapa de prototipação foi utilizado um UI Kit do Figma, que foi customizado, de acordo com o projeto desenvolvido.

A cliente viu os modelos e escolheu o que mais gostou.

05. React Native & Expo

Instalações do React Native e Expo no pc e do Expo Go no celular.

Foram feitos estudos aprofundados para compreensão das possíveis arquiteturas a serem construídas.

Achei muito empolgante gerar um QRCode a partir da pasta do projeto no pc que, quando lido pelo celular, (com o Expo Go previamente instalado), o aplicativo construído aparece no celular! Legal demais!

Outro ponto importante é o fato de que é possível alterar o código no pc e visualizar as mudanças no celular automaticamente.

Conclusão

Foi um projeto empolgante, cheio de desafios, novidades e aprendizagem.

Não foi fácil, pois o desenvolvimento foi complexo, mas valeu cada lágrima e as poucas horas de sono.

O aplicativo teve a estruturação e telas finalizadas. A próxima etapa será o desenvolvimento do PDV (ponto de venda).

A integração com o Banco de Dados foi feita com o Firebase.

Em aplicativos e-commerce, é necessário que a parte de segurança seja muito bem elaborada, pois envolve dados pessoais dos usuários e transações bancárias.

Existem infinitas maneiras de construir um aplicativo e-commerce, diversos frameworks e tecnologias.

Em 2025, se as próximas etapas forem concluídas, o aplicativo estará disponível nas App Stores.

Batalha dos Pets

Mobile Project - Desafio UX/UI

Design do App Mobile "Batalha dos Pets" como desafio da Trilha "Introdução ao Design UI/UX".

Desafio & Objetivo

O desafio se baseia na criação do design de um aplicativo, onde serão postadas fotos de pets diariamente, sem edição, para serem votadas por todos os participantes.

Os 5 pets mais votados se consagram campeões do dia.

O principal objetivo deste projeto foi incentivar as alunas da trilha a colocarem em prática os conhecimentos adquiridos durante o percurso dos módulos do curso.

Para o desenvolvimento das etapas do desafio, as alunas utilizaram a plataforma Figma.
