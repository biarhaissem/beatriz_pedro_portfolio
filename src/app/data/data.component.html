Dashboard Gerencial

Análise de Dados com Power BI

Dashboard Gerencial para monitoração de fábrica de roupas.

Desafio & Objetivo

O objetivo deste desafio era a criação de um dashboard gerencial para análise de dados fornecidos de uma fábrica de roupas fictícia.

O desafio englobava a identificação de gargalos operacionais, performance e oportunidades de melhoria.

Os filtros exigidos eram: Data, Depto e Equipe.

KPI: Produção Real, Meta, Desvio, Ociosidade e Incentivo.

01. Data Cleansing

A partir dos dados disponibilizados em Excel, foram identificados os erros e inconsistências para realizar a análise adequadamente.

Essa etapa é muito importante pois somente com dados confiáveis podemos fazer uma análise correta.

Utilizei o Power Query, do Power BI, para facilitar as correções e abri o arquivo para iniciar a criação das fórmulas.

02. Fórmulas DAX

A criação das fórmulas, utilizando DAX, foi a etapa seguinte do projeto.

Nessa etapa, é necessário levar em consideração quais os pontos mais importantes que devem ser analisados para a organização lógica do projeto.

Após criarmos as fórmulas, inicia-se a criação do layout e a escolha de quais tipos de gráficos utilizaremos para que a compreensão dos dados fique intuitiva e agradável.

03. Design do Dashboard

A etapa de Design do Dashboard é o momento em que a lógica encontra a criatividade.

Para buscar inspiração, vale a pena pesquisar exemplos disponíveis na internet para encontrar um possível caminho.

Existem inúmeras maneiras de diagramar um dashboard, não existe uma regra rígida.

Porém, é imprescindível ter consciência que a leitura do dashboard deve ser intuitiva e de fácil compreensão para ter a utilidade almejada.

04. Plataformas e Ferramentas

Para conseguirmos chegar ao objetivo visual almejado, podemos utilizar diversas plataformas existentes.

Eu gosto de utilizar o Figma e o Canva, pois consigo criar a "tela de fundo" para o dashboard e adicionar ao layout.

Podemos testar algumas alternativas até encontrarmos a que irá satisfazer o trabalho.

Só é preciso ter cuidado para não "viajar" demais e perder o prazo da entrega, rsrsrs!
