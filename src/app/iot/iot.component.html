PLANTECH

IoT Mobile App - Monitor de Umidade

App mobile IoT para monitoramento remoto da umidade dos vasos de plantas.

Desenvolvemos um protótipo IoT utilizando a placa programável Wi-Fi ESP8266, um sensor de umidade e a IDE Arduino para programação com a linguagem C.

Através da plataforma ucraniana Blynk IoT, criamos a interface para controle remoto mobile integrado ao protótipo para receber os dados em tempo real.

Utilizamos ferramentas modernas para a elaboração e desenvolvimento do projeto.

Smart Plant

IoT Mobile App - Monitoração e Irrigação Remotas

Pudemos utilizar alguns componentes do PLANTECH para integrar ao sistema remoto de irrigação sustentável.

Criamos outro aplicativo mobile pela Blynk IoT.
